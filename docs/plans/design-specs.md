## 1. 整体设计风格

### 1.1 参考设计
- 参考网站：https://object-x.com.cn/
- 风格定位：现代简洁，侧边栏 + 内容区布局

### 1.2 布局结构

**桌面端布局：**
```
┌─────────────┬──────────────────────────────┐
│             │                              │
│   侧边栏    │        主内容区               │
│  (固定)     │                              │
│             │                              │
│   250-300px │         自适应宽度            │
│             │                              │
└─────────────┴──────────────────────────────┘
```

**移动端布局：**
- 侧边栏隐藏，通过右下角悬浮按钮（汉堡菜单）唤出
- 点击后从左侧滑出抽屉式菜单
- 主内容区占满全屏

## 2. 配色方案

### 2.1 浅色模式（Light Mode）
```css
--bg-main: #F9F4E1;        /* 主背景 - 米黄色 */
--bg-sidebar: #2B2620;     /* 侧边栏 - 深棕色 */
--text-primary: #3A352E;   /* 主文字 - 深棕色 */
--accent: #7E8C6A;         /* 点缀色 - 橄榄绿 */
```

### 2.2 深色模式（Dark Mode）
```css
--bg-main: #2B2620;        /* 主背景 - 深棕色 */
--bg-sidebar: #1A1612;     /* 侧边栏 - 更深的棕黑色 */
--text-primary: #E8E2CF;   /* 主文字 - 米白色 */
--accent: #8FA07C;         /* 点缀色 - 浅橄榄绿 */
```

### 2.3 配色应用
- **按钮、链接、高亮**：使用点缀色
- **卡片、输入框**：使用主背景色的变体（稍深或稍浅）
- **边框、分割线**：使用文字色的低透明度版本

   - 位置：页面顶部固定

   - 桌面端：右侧固定侧边栏

   - 位置：文章头部元信息区

## 6. 响应式设计

### 6.1 断点

```css
/* 手机 */
@media (max-width: 640px) { }

/* 平板 */
@media (min-width: 641px) and (max-width: 1024px) { }

/* 桌面 */
@media (min-width: 1025px) { }
```

### 6.2 布局适配

**侧边栏：**
- 桌面：固定左侧，250-300px 宽
- 平板：同桌面
- 手机：隐藏，通过汉堡菜单唤出

**文章列表：**
- 桌面：单列列表
- 平板：单列列表
- 手机：单列列表（缩略图变小）

**文章详情：**
- 桌面：内容区 + 右侧 TOC
- 平板：内容区 + 右侧 TOC（TOC 宽度减小）
- 手机：仅内容区，TOC 折叠到顶部

**字体大小：**
- 桌面：正文 16-18px
- 手机：正文 14-16px

## 8. 动画和交互

### 8.1 过渡动画

**通用原则：**
- 使用 CSS transition
- 时长：200-300ms
- 缓动函数：ease-in-out

**应用场景：**
- 按钮悬停
- 卡片悬停
- 主题切换
- 侧边栏展开/收起

### 8.2 滚动行为

**平滑滚动：**
```css
html {
  scroll-behavior: smooth;
}
```

**回到顶部：**
- 平滑滚动到页面顶部
- 动画时长：500ms

**目录导航：**
- 点击后平滑滚动到对应标题
- 当前阅读位置高亮
